# Sudoku

The game was built as part of she codes Python course.
The program takes a simple text file called board.txt. 
This text file should contain 9 columns and 9 row of numbers used for a template to the initial board game. 
Zero is used to indicate empty square. To win the game the user need to fill the empty squares so that the
numbers 1-9 appear exactly once in each row, column or 3X3 square. The 3X3 squares surrounded by blue lines.
The user can reset the game to the intail board or undo as many moves as needed.
